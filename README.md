# Bulbul Restaurant — Flutter Web (UI Prototype)

A responsive single-page restaurant website built entirely with Flutter Web. Designed to demonstrate Flutter's capability as a cross-platform framework for web UI development.

---

## Sections

- **Header** — Navigation bar with restaurant branding
- **Hero** — Full-width banner with call-to-action
- **Menu** — Food items display with clean card layout
- **Reviews** — Customer testimonials section
- **Map** — Embedded map section for restaurant location
- **Footer** — Contact info and links

---

## Highlights

- Built with Flutter Web — demonstrating cross-platform capability
- Fully responsive layout — adapts to different screen sizes
- Single-page smooth scroll experience
- Clean and modern restaurant UI

---

## Tech Stack

| Layer | Technology |
|---|---|
| Framework | Flutter Web |
| Language | Dart |
| UI | Custom widgets, responsive layout |
| Map | Flutter map widget |

---

## Getting Started

1. Clone the repository
```
git clone https://github.com/FiazZafar/bulbul-website.git
```
2. Run `flutter pub get`
3. Run with `flutter run -d chrome`
